# Malt Mafia Website

Welcome to the Malt Mafia website! This playful site is dedicated to our love for malts and Handles ice cream. Here, you can explore the rich history of malts and learn about the delightful world of Handles ice cream.

## Project Structure

The project is organized as follows:

```
malt-mafia-website
├── src
│   ├── index.html          # Main entry point of the website
│   ├── about-maltes.html   # Page about the history of malts
│   ├── about-handles.html  # Page about the history of Handles ice cream
│   ├── styles
│   │   └── style.css       # CSS styles for the website
│   └── scripts
│       └── app.js          # JavaScript for interactive elements
├── assets
│   ├── images              # Directory for image files
│   └── fonts               # Directory for font files
├── package.json            # npm configuration file
└── README.md               # Documentation for the project
```

## Features

- **Home Page**: A vibrant introduction featuring a filler image of a Handles malt.
- **History of Malts**: Discover the origins and evolution of malts.
- **History of Handles Ice Cream**: Learn about the delightful journey of Handles ice cream.

## Getting Started

To get started with the Malt Mafia website, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd malt-mafia-website
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open `src/index.html` in your web browser to view the website.
